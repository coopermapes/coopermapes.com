Tuning Notes & Assumptions

Values reflect deviation from equal temperament in cents, derived from just intonation ratios.

  • Chord tones are the parent scale's notes stacked in thirds (root–3–5–7–9–11–13). No chromatic alterations introduced.
  • Minor keys use natural minor (Aeolian): i, ii°, III, iv, v, VI, VII, with no raised leading tone.
  • Intervals with separate major/minor values (♭13, 13, ♭7, maj7) use the value matching that chord's own quality. Diminished chords use the minor value.
  • Enharmonic keys follow common-practice spelling (D♭ over C♯ for major, etc.).
  • Chromatic tones (via toggle) are spelled from each chord's own stacked-third position, so a key can call for a double flat or double sharp instead of defaulting to the simplest enharmonic option.
